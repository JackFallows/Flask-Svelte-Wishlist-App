# Wishlist web app written in Python/Flask with a Svelte frontend

Build client code with:
```
./node_modules/.bin/gulp
```

Run Playwright component tests with:
```
npm run test-ct
```

Run Jasmine module unit tests with:
```
npm run test-m
```
